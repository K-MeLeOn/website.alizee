Contraintes techniques :

HTML
o Le code est indenté et lisible
o Toutes les images sont dans un dossier « images »
o Le formulaire envois vers un Mailto :…@
CSS
o Une feuille de style dans un fichier séparé
o Les styles de l’en-tête, menu, pied de page, fond de page, doivent pouvoir se modifier en une ligne : cad, par exemple, Une modification de la couleur de titre de l’en-tête change cette couleur sur toutes les pages.
o Au survol des liens, leur couleur change
o Les 3 sortes de CSS sont utilisées : redéfinition d’une balise, class et ID
JAVASCRIPT / JQUERY
o Intégration par le CDN Google ou Microsoft
